## üîß US-008 : Configure Frontend to Call Laravel API in Dev

**Role:** Full-stack developer with access to the `photosync` repo (Laravel backend + Vue frontend).

**Goal:** Make the Vue Contact Us page send requests to the local Laravel API (`/api/messages`) during development.

---

### ‚úÖ Task Description

Wire the Vue frontend to the Laravel backend in **development environment** so that the Contact Us form successfully sends POST requests to:

```text
http://localhost:8000/api/messages
```

and receives a `201 Created` response when valid data is submitted.

---

### üß© Steps for the AI Agent

1. **Ensure Laravel dev server config**

   * In the Laravel project (`photosync` root), make sure the app can run with:

     ```bash
     php artisan serve --host=0.0.0.0 --port=8000
     ```
   * No code changes needed unless there‚Äôs a port conflict or binding issue.

2. **Configure frontend API base URL via Vite env**

   * In the Vue app (folder: `frontend/`), create or update the file:

     ```text
     frontend/.env
     ```
   * Add or update:

     ```env
     VITE_API_URL=http://localhost:8000
     ```

3. **Verify Contact Us page uses `VITE_API_URL`**

   * Open:

     ```text
     frontend/src/pages/Contact.vue
     ```
   * Confirm the `fetch` call uses `import.meta.env.VITE_API_URL` as base, like:

     ```js
     const res = await fetch(import.meta.env.VITE_API_URL + "/api/messages", {
       method: "POST",
       headers: { "Content-Type": "application/json" },
       body: JSON.stringify({ email: this.email, msg: this.msg }),
     });
     ```

4. **Run dev servers and validate (locally)**

   * Backend:

     ```bash
     php artisan serve --host=0.0.0.0 --port=8000
     ```
   * Frontend (in `frontend/`):

     ```bash
     npm run dev
     ```
   * From the browser:

     * Open the frontend dev URL (e.g., `http://localhost:5173`).
     * Go to **Contact Us** page.
     * Submit a test message.

5. **Expected behavior**

   * Network tab in devtools should show:

     * `Request URL`: `http://localhost:8000/api/messages`
     * `Method`: `POST`
     * `Status`: `201 Created` (if backend is configured correctly)
   * Frontend should show a success message (e.g., ‚ÄúMessage sent successfully!‚Äù).

6. **Commit changes**

   * If any files were changed (e.g., `.env`, `Contact.vue`), create a commit:

     * Message suggestion:

       ```text
       Configure frontend to use Laravel API in dev via VITE_API_URL
       ```
