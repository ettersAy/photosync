## Goal
Transform GitHub repository into Laravel 12 backend project.

---

## Actions & Commands

- Navigate to repository: `$ cd ~/photosync`

- Confirm location: `$ pwd`

- Remove existing README: `$ rm README.md`

- Install Laravel 12 in current directory: `$ composer create-project laravel/laravel . "12.*"`

- Verify installation: `$ php artisan --version`

- Commit changes to GitHub:
  ```bash
  $ git status
  $ git add .
  $ git commit -m "Install Laravel 12 backend"
  $ git push origin main
  ```
