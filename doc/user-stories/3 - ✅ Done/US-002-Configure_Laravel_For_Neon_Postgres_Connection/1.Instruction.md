We will:

1. Create your `.env` file
2. Insert your Neon credentials
3. Test the DB connection with `php artisan migrate`

---

## Actions & Commands

- In Cloud Shell (inside `~/photosync`): `$ cp .env.example .env`

- Generate App Key: `$ php artisan key:generate`

- Edit `.env` with your Neon connection values, Find the DB section and replace it so it looks like:

```
DB_CONNECTION=pgsql
DB_HOST=<your-neon-host>
DB_PORT=5432
DB_DATABASE=photosync_db
DB_USERNAME=<your-neon-username>
DB_PASSWORD=<your-neon-password>
DB_SSLMODE=require
```

‚ö†Ô∏è **Important:** Add to `.env` this line manually: `DB_SSLMODE=require` Because Neon **requires SSL**.

- Test DB Connection, Run migrations (these are the default Laravel tables): `$ php artisan migrate`
