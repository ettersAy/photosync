Edit `database/migrations/*_create_messages_table.php` with:

```php
Schema::create('messages', function (Blueprint $table) {
    $table->id();
    $table->string('email')->nullable();
    $table->text('msg');
    $table->timestamps();
});
```

### Execute Migration
`$ php artisan migrate --database=pgsql_direct`
